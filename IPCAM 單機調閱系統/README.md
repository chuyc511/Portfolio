# IPCAM 單機調閱系統 (Web Django)
使用 Bootstrap Modal 做彈出視窗設定 ，後端用 ffmpeg 將 IPCAM RTSP流擷取下來並製作 m3u8 列表，前端則是用 Video.js 播放該 m3u8 列表。
